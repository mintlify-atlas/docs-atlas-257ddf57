IPEX-LLM is a PyTorch library from Intel for running large language models with very low latency on Intel hardware — including iGPUs, discrete Arc GPUs, NPUs, and CPUs. With a single line of code change, you get INT4/INT5/INT8/FP4/FP6/FP8 quantized inference, multi-GPU serving, and finetuning on the same hardware you already own.

## Get running in minutes

<Steps>
  <Step title="Install IPEX-LLM">
    Install via pip for your platform (GPU or CPU):
    ```bash
    pip install --pre --upgrade ipex-llm[xpu] --extra-index-url https://pytorch-extension.intel.com/release-whl/stable/xpu/us/
    ```
  </Step>
  <Step title="Load a model with INT4 optimization">
    Use the familiar HuggingFace API with one additional parameter:
    ```python
    from ipex_llm.transformers import AutoModelForCausalLM
    from transformers import AutoTokenizer

    model = AutoModelForCausalLM.from_pretrained(
        "meta-llama/Llama-2-7b-chat-hf",
        load_in_4bit=True
    )
    tokenizer = AutoTokenizer.from_pretrained("meta-llama/Llama-2-7b-chat-hf")
    ```
  </Step>
  <Step title="Run inference on Intel GPU">
    Move the model to XPU and generate:
    ```python
    import torch

    model = model.to('xpu')

    with torch.inference_mode():
        input_ids = tokenizer.encode("What is Intel?", return_tensors="pt").to('xpu')
        output = model.generate(input_ids, max_new_tokens=64)
        print(tokenizer.decode(output[0], skip_special_tokens=True))
    ```
  </Step>
</Steps>
