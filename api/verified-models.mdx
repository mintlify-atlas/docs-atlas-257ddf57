## How to load a verified model

All verified models are loaded through the same `AutoModel` API. Use the HuggingFace model ID directly:

```python
from ipex_llm.transformers import AutoModelForCausalLM
from transformers import AutoTokenizer

# Example: Qwen2-7B on GPU in INT4
model = AutoModelForCausalLM.from_pretrained(
    "Qwen/Qwen2-7B-Instruct",
    load_in_4bit=True,
    cpu_embedding=True,
    optimize_model=True,
    trust_remote_code=True,
)
model = model.half().to("xpu")

tokenizer = AutoTokenizer.from_pretrained("Qwen/Qwen2-7B-Instruct", trust_remote_code=True)
```

<Note>
Models that use custom code (ChatGLM, Qwen, Baichuan, InternLM, etc.) require `trust_remote_code=True` in both `from_pretrained` calls.
</Note>
